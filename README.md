# 🦠 India COVID-19 State-wise Dashboard (Excel)

An interactive Excel dashboard analyzing COVID-19 data across all Indian states and union territories. Built as part of my data analyst portfolio to demonstrate skills in data cleaning, pivot tables, and dashboard design.

## 📁 Project Structure

| Sheet | Description |
|---|---|
| `Dashboard` | Main summary with KPIs — Top/Bottom 5 states by Cases, Recovery, Deaths & Active |
| `Sheet1` | Cleaned raw data with all states, zones, ratios and above/below average flags |
| `Total Cases` | Pivot – Top 5 states by total cases |
| `Recovered` | Pivot – Bottom 5 states by recovery rate |
| `Death` | Pivot – Bottom 5 states by death rate |
| `Active` | Pivot – States with highest active case % |
| `Active & Death Cases` | Combined state-wise active and death count |
| `Cases by Zone` | Regional breakdown – East, West, North, South |
| `Total & Discharged` | State-wise total vs discharged comparison |
| `Death Average` | State-wise death ratio with national average |
| `Recovered Average` | State-wise recovery ratio with national average |

---

## 🔍 Key Insights

- **Maharashtra** recorded the highest total cases (66.2 lakh), followed by Kerala (50.6 lakh) and Karnataka (29.9 lakh)
- **Punjab** had the highest death rate at 2.75%, despite not being among the top states for total cases
- **Daman and Diu** had the lowest death rate (0.04%) and the highest recovery rate (99.96%)
- **Mizoram** had the highest active case percentage (4.39%) and the lowest recovery rate (95.25%)
- **South India** accounted for ~39.6% of total cases nationally — the highest among all four zones
- Nationally, **98.36%** of patients recovered on average
